Export to 3000×3000 PNG

  1. Screenshot: Press Cmd + Shift + 4 → Space → click the cover
  2. Open in Preview: Double-click the screenshot from Desktop
  3. Upscale: Tools → Adjust Size → 3000 px wide → Scale proportionally + Resample image both checked → OK
  4. Save as PNG: File → Export → PNG → save as toroiq_cover_v1.png
  5. Upload to Spotify in the cover art field
